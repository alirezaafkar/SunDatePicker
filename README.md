## Getting started

### Dependency

```
dependencies {
    implementation 'com.alirezaafkar:sundatepicker:2.1.2'
}
```

### Usage

```java
 new DatePicker.Builder()
            .id(id)
            .minDate(minDate)
            .maxDate(minDate)
            .date(initialDate)
            .build(MainActivity.this)
            .show(getSupportFragmentManager(), "");
```
